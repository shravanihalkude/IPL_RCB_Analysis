# IPL_RCB_Analysis
This project analyzes IPL player performance and auction data to help Royal Challengers Bengaluru (RCB) identify high-performing and cost-effective players for team selection. The analysis focuses on optimizing auction decisions by evaluating player performance, consistency, and value for money.
# Business Problem
RCB aims to build a championship-winning squad while managing auction budgets effectively. The objective is to identify players who deliver strong on-field performance and provide maximum value relative to their auction price.
# Objectives
Analyze player performance across IPL seasons.
Identify top-performing batsmen and bowlers.
Evaluate player consistency and reliability.
Compare player performance against auction prices.
Discover undervalued players who offer high returns on investment.
Provide data-driven recommendations for future auction strategies.
